Portofolio

Bookshelf Website

Boku Bookshelf sebuah website yang digunakan untuk menyimpan data-data buku, dengan fitur-fitur untuk menambahkan buku, mengedit, dan menghapus buku, selain itu web ini juga memiliki fitur login & register dengan autentikasi menggunakan JWT. Proyek ini dikembangkan dengan menggunakan React untuk front-end, Node.js untuk back-end, dan database MariaDB.

Tech Stack :

  • Front-end : React JS
  • Back-end : Node.js dengan framework Hapi
  • Database : MariaDB
  • Lainnya : Tailwind CSS, Chakra UI, jsonwebtoken, bcrypt

Repository : https://github.com/riyandrmd/boku-bookshelf-apps.git

Ngotaku Anime Website

Website info anime dengan tampilan responsif, terinspirasi dari website MyAnimeList, menampilkan daftar anime sedang atau akan tayang, serta trailer anime terbaru. Situs web ini dibuat menggunakan HTML, CSS, dan JavaScript dengan mengintegrasikan API MyAnimeList dari jikan.moe.

Tech Stack :

  • fontawesome
  • API source : https://jikan.moe/
